# Stopwatch

A stopwatch that displays start, stop, and reset buttons.

## What it does

- Shows the hours, minutes, and seconds
- Updates without reloading the page
-  When start is clicked, the time will start; when stop is clicked the time will stop; and when reset is clicked the time will display 00:00:00

## Run locally

Download foldder and open [index.html] in any browser
